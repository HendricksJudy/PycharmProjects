# 数据类型详解—列表
> 列表是一组有序的数据组合，列表中的数据可以被修改

# 列表定义
+ 可以使用 [] 中括号进行定义
+ 可以使用 list函数 定义
+ 在定义列表中的元素时， 需要在每个元素之间使用 , 逗号 进行分隔
+ 列表中的元素可以是任意类型，通常用于存放同类项目的集合
